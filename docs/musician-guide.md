# Musician Guide — wstprtradio

Welcome! This guide explains how to go live on wstprtradio as a musician or DJ.

## Option 1: Web DJ (Easiest — Audio Only)

No software needed. Works in Chrome or Firefox.

1. Ask an admin to open a Web DJ session for you (or if you have admin access, go to **Admin → Live → Open Web DJ**)
2. You'll get a unique URL — open it in your browser
3. Allow microphone access when prompted
4. Click **Start Broadcasting** — you're live!
5. Click **Stop Broadcasting** when done

**Tip:** Use headphones to avoid feedback. The stream delay is ~10–15 seconds.

## Option 2: BUTT (Broadcast Using This Tool) — Audio Only

Good for: music playback from your computer, mixing software output.

See [butt-fallback-guide.md](./butt-fallback-guide.md) for full BUTT setup instructions.

## Option 3: OBS Studio — Video + Audio

For video live streams (shown on the Live page).

See [obs-profile-guide.md](./obs-profile-guide.md) for full OBS setup.

## Stream Details

| Setting | Value |
|---|---|
| Server | Provided by admin |
| Port | 8000 (Icecast) |
| Mount point | `/radio` |
| Format | MP3 128kbps or AAC |

## Tips for a Good Stream

- **Use wired internet** if possible
- **Set your sample rate to 44100 Hz** in your audio interface
- Test your stream a few minutes before going live
- Communicate with the admin before your session
