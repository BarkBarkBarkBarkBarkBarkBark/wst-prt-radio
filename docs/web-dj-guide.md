# Web DJ Guide — wstprtradio

The AzuraCast Web DJ allows you to broadcast directly from your browser — no extra software needed.

## Requirements

- Chrome 80+ or Firefox 75+
- A working microphone (USB or built-in)
- Stable internet connection (5+ Mbps upload recommended)

## Step-by-Step

### As a DJ (without admin access)

1. Contact the station admin to schedule your slot
2. The admin will generate a Web DJ link and send it to you
3. Open the link in Chrome or Firefox
4. Click **Allow** when the browser asks for microphone access
5. Select your audio input device from the dropdown
6. Set your bitrate (128kbps MP3 recommended)
7. Click **Start Broadcasting**
8. Monitor your levels — keep them in the green/yellow range
9. Click **Stop Broadcasting** when your set is done

### As an Admin

1. Go to **Admin Panel → Live → Open Web DJ**
2. The system will open the AzuraCast Web DJ interface in a new tab
3. Share the URL with your guest DJ if needed

## Audio Tips

- **Monitor**: Use headphones — the stream has ~10–15s delay
- **Levels**: Aim for -12 to -6 dBFS average, peaks at -3 dBFS
- **Format**: Use MP3 128kbps for compatibility

## Troubleshooting

**"Microphone not found"**: Check browser permissions (click the lock icon in the address bar)

**Stream cutting out**: You may have connection issues — try wired ethernet

**No sound on the website**: Give it 15–20 seconds after starting — there's a buffer delay
